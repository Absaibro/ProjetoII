# Notebook de Machine Learning

Esta pasta contém o notebook utilizado para preparação dos dados, análise exploratória (EDA), treinamento e avaliação dos modelos de Machine Learning desenvolvidos para o projeto.

Etapas desenvolvidas:

- Importação dos dados;
- Tratamento dos dados;
- Engenharia de atributos;
- Análise Exploratória (EDA);
- Treinamento dos modelos;
- Avaliação dos resultados.
